<국가별 k-컨텐츠 선호도 분석>
: 
- 컨텐츠 선호도(대중적 인기)순위를 파악함
    barplot을 그려 어떠한 컨텐츠가 대중적으로 인기있는지를 파악해봄
    각 컨텐츠 별로의 mean값을 구해 barplot을 그림
    --> 해당 파악을 통해 어떤 컨텐츠 순으로 대중적인 인기가 있는지를 파악함
- 아시아국가를 기준으로 데이터를 분리해 년도별 아시아 국가간의 컨텐츠 선호도 차이가 있는지 파악
    lineplot을 그려보았을때 시각적으로 년도별 국가가간의 컨텐츠 선호도 차이가 있는 것으로 보임
    정확한 파악을 위해 welch_anova test를 진행함
    p-value 값이 0.05보다 작으므로 귀무가설 기각
    --> 아시아 국가간의 선호도 차이가 있는 것으로 파악됨
- 비아시아국가를 기준으로 데이터를 분석해 년도별 비아시아 국가간의 컨텐츠 선호도 차이가 있는지 파악
    * 분석 진행 필요*
- 아시아 국가와 비아시아 국가간의 선호도 차이 분석
    아시아 국가와 비아시아 국가간의 선호도 차이가 있는지를 파악하기 위해 Welch's t test 수행
    p-value가 0.05보다 작으므로 귀무가설 기각
    아시아와 비아시아 국가 집단 간에는 유의미한 선호도 차이가 존재함
- 성별간 선호도 총합과의 상관관계 분석
    선호도 총합에 여성과 남성 중 어떤 성별이 더 많은 영향을 주는지 확인해보기 위해 여성과 남성을 구분하여 선호도총합간의 상관분석 실시
    여성과 선호도 총합간의 scatter 그림/남성과 선호도 총합간의 scatter 그림
    --> 양의 상관관계가 있는 것으로 파악됨
    각각 피어슨 상관분석 실행
    --> 선호도 총합과 여성간의 상관계수값이 선호도 총합과 남성간의 상관계수값보다 더 작으므로, 여성이 남성에 비해 선호도 총합에 더 강햔 양적관계를 가지고 있다는 것을 알 수 있음
- 연령대별 선호도 차이 분석
    연령대별로 선호도 차이를 파악하기 위해 barplot그림
    선호도 차이가 유의미한지를 파악해보기 위해 anova 분석 진행
    10~50대 집단 모두 anova 분석을 실행한 결과 p-value 값이 0.05보다 매우 작음
    --> 유의미한 차이를 지닌 집단이 존재한다.
    * 유의미한 차이가 있기 때문에 사후검정 진행(bonferroni,Tukey HSD 진행) --> 해당 분석 필요 *
    <밑에 내용은 사후검정을 통해서 밝혀내기>
    그러나 10~30대 집단 간의 anova 분석 결과 p-value 값이 0.054으로 유의미한 차이는 나지 않는 것을 확인했다.
    30~50대 집단 간의 anova 분석 결과 p-value 값이 0.05보다 매우 작으므로 유의미한 차이를 지닌 집단이 존재한다.
    결론: 10~30대는 서로 비슷한 높은 선호도를 보여주었지만 4~50대는 그보다 유의미하게 낮은 선호도를 보여주었다.
